# Student Performance Analysis Using Pandas

## Overview

This project analyzes student performance data using the Pandas library in Python. The application reads student records from a CSV file, calculates total scores and percentages, and generates meaningful insights about student performance.

## Features

* Load student data from a CSV file
* Calculate Total Score for each student
* Calculate Percentage for each student
* Identify the Top Performer
* Identify the Bottom Performer
* Calculate average scores for Math, Reading, and Writing
* Find highest and lowest scores in each subject
* Count total students by gender
* Calculate average percentage by gender
* Find the highest scorer from each gender
* Display the Top 10 students based on percentage
* Export the processed data to a new CSV file

## Technologies Used

* Python
* Pandas

## Dataset

The project uses the `Students_Performance.csv` dataset containing:

* Gender
* Math Score
* Reading Score
* Writing Score
* Other student-related information

## Calculations Performed

### Total Score

Total Score = Math Score + Reading Score + Writing Score

### Percentage

Percentage = (Total Score / 300) × 100

## Output Generated

The program provides:

1. Complete student dataset with Total Score and Percentage.
2. Top performer details.
3. Bottom performer details.
4. Average score for each subject.
5. Highest and lowest scores for each subject.
6. Gender-wise student count.
7. Gender-wise average percentage.
8. Highest scorer from each gender.
9. Top 10 students based on percentage.
10. Processed report exported as `student_data_report.csv`.
